#blocks 

Maps can be created using the [Bases map view](https://obsidian.md/help/bases/views/map).

Maps are compatible with [[Block width]] features.

## Helper classes

Use [[Helper classes]] to control map appearance on a specific note:

| Class      | Description                       |
| ---------- | --------------------------------- |
| `map-100`  | Maps fill 100% of the pane width  |
| `map-max`  | Maps fill the max line width      |
| `map-wide` | Maps fill the wide line width     |
